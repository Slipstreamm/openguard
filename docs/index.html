AI Mod Bot Documentation

User Information (`abtuser.py`)

/aboutuser [user]
Displays detailed information about a specified user or the person who issued the command.
/userinfo_admin set [user] [key] [value]
Sets a custom data field for a user (admin only).
/userinfo_admin remove [user] [key]
Removes a custom data field from a user (admin only).
/userinfo_admin list [user]
Lists all custom data fields for a user (admin only).
/userinfo_admin clear [user]
Clears all custom data for a user (admin only).

AI Moderation (`aimod.py`)

/aimod config logchannel [channel]
Sets the channel for moderation logs.
/aimod config setlang [language]
Sets the bot's response language for the server.
/aimod config suggestionschannel [channel]
Sets the channel for user suggestions.
/aimod config moderatorrole [role]
Sets the role for moderators.
/aimod config suicidalpingrole [role]
Sets the role to be pinged for suicidal content alerts.
/aimod config addnsfwchannel [channel]
Adds a channel to the NSFW list.
/aimod config removensfwchannel [channel]
Removes a channel from the NSFW list.
/aimod config listnsfwchannels
Lists all configured NSFW channels.
/aimod config enable [true/false]
Enables or disables AI moderation for the server.
/aimod infractions view [user]
Views a user's infraction history.
/aimod infractions clear [user]
Clears a user's infraction history.
/aimod model set [model]
Sets the AI model for moderation.
/aimod model get
Displays the current AI moderation model.
/aimod debug testmode [true/false]
Enables or disables test mode for AI moderation.
/aimod debug last_decisions
Shows the last 5 AI moderation decisions.
/aimod appeals appeal [reason]
Submits an appeal for a moderation action.
/pull_rules
Updates the server's rules from the designated rules channel.
/aimod globalban manage [add/remove] [userid]
Adds or removes a user from the global ban list.

Credits (`credits.py`)

/aimodcredits
Displays the credits for the AI Mod Bot.

Hardware Information (`hwinfo.py`)

/system check
Shows detailed system and bot information.
/system temps
Displays hardware temperature information.

Information (`info.py`)

/aimodinfo help
Shows a list of all `/aimod` commands.

Shell (`shell.py`)

/shell [command]
Executes a shell command (owner only).
/shell_app [command]
Executes a shell command (owner only).

Statistics (`statistics.py`)

This module provides bot statistics but has no user-facing commands.
